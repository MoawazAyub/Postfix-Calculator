Postfix Calculator

A simple Calculator Project.

How to get started:

Load the .sln file in visual studio to run the project.
This project first converts the input expression into postfix expression for example 3 + 4 is converted to 3 4 + and then calculate the result.
You can read more about postfix or prefix from here
